# Marion's Forgiveness Countdown

A lightweight static website that displays a countdown timer showing the time remaining until November 8, 2034 at 12:00 PM Melbourne time.

## Features

- Real-time countdown that updates every second
- Displays hours, minutes, and seconds remaining
- Clean, modern design with gradient background
- Fully responsive (works on mobile and desktop)
- Single HTML file - no dependencies or build process required
- Works offline once loaded

## Deployment

This is a static website that can be deployed to any free hosting platform. Here are some popular options:

### GitHub Pages

1. Create a new repository on GitHub
2. Push the `index.html` file to the repository
3. Go to Settings â†’ Pages
4. Select the branch (usually `main` or `master`)
5. Your site will be available at `https://yourusername.github.io/repository-name`

### Netlify

1. Go to [netlify.com](https://netlify.com)
2. Sign up or log in
3. Drag and drop the `index.html` file onto the Netlify dashboard
4. Your site will be live instantly with a free `.netlify.app` domain

### Vercel

1. Go to [vercel.com](https://vercel.com)
2. Sign up or log in
3. Click "New Project"
4. Import your Git repository or upload the `index.html` file
5. Deploy - your site will be live in seconds

### Other Options

- **Surge.sh**: `surge` command-line tool for instant deployments
- **Cloudflare Pages**: Free static site hosting
- **Firebase Hosting**: Google's free hosting service
- Any web server that can serve static HTML files

## Local Development

Simply open `index.html` in a web browser. No server or build process required.

- Pure HTML, CSS, and JavaScript
- No external dependencies
